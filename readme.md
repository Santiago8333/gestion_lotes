Proyecto de gestión de lotes y recibos
La pagina permitirá agregar lotes y crear recibos de persona física y jurídica y los usuarios son administrados por un administrador los administradores solo ellos pueden registrar empleados y eliminar entidades

Administrador: acceso completo; crea empleados y elimina entidades.

Empleado: acceso limitado; puede gestionar lotes y recibos, pero no eliminar registros críticos ni administrar usuarios.

los usuarios podrán crear recibos y modificarlos, pero no podrán eliminarlos 
para ingresar al sistema el administrador deberá crear una cuenta de empleado
y ingresar con la contraseña


se podrá agregar varias una forma de pago para recibo de forma física y jurídica
